Week of 2026-03-13

# 🤖 This Week in AI: March 9-12, 2026 ## Top Developments **GPT-5.4 Launches with Computer Control & 1M Token Context** OpenAI released its latest model with native ability to control computers (click, type, read screens), supports up to 1 million tokens of context, and excels at spreadsheets and knowledge work. The model is rolling out across ChatGPT, API, and Codex with multiple speed tiers including a new `/fast` mode. **Claude Adds Code Review & Remote Control Features** Anthropic deployed multiple AI agents that automatically review pull requests for bugs, plus a Remote Control feature letting you start tasks on desktop and manage them from your phone. Meanwhile, Claude committed to staying completely ad-free while competitors explore monetization. **Karpathy Open-Sources AI Research Automation** Tesla's former AI lead released "autoresearch"—a system where AI agents autonomously improve their own code over time, plus a complete GPT implementation in just 243 lines of Python. His nanochat now trains GPT-2 in 2 hours on a single GPU. **OpenAI Deploys Models to Department of War** OpenAI announced an agreement to deploy AI models on classified military networks, marking significant expansion into government defense. CEO Sam Altman emphasized safety partnerships and fielded questions about the decision. **WiFi-Based Body Tracking Goes Open Source** Researchers open-sourced WiFi-DensePose, technology that maps a person's body position through walls using only WiFi signals—no cameras needed. The release raises both innovation and privacy concerns. **Programming Has Fundamentally Changed in December** Andrej Karpathy emphasized that AI coding agents crossed a critical capability threshold last month and "basically didn't work before December"—the shift happened suddenly, not gradually, changing how developers work daily. --- ## 🔥 Trend of the Week: The Great Unbundling of AI Labor The week revealed a pattern: AI is rapidly automating specific high-value jobs rather than replacing entire roles. Claude found 22 Firefox vulnerabilities in two weeks (work that required security specialists). GPT-5.4 handles spreadsheets like finance analysts. Automated code review replaces expensive senior developer time. The common thread? These aren't general replacements—they're surgical replacements of expensive *specialist tasks*. This means layoffs are coming for junior analysts, junior developers, and entry-level consultants, but the disruption follows specific job functions, not job titles. Companies are learning to "hire AI + one person" instead of "hire five people." --- ## 💡 Try It Yourself
  • Build Your Own "Autoresearch" Loop: Use Karpathy's nanochat code to set up a simple script where Claude or GPT edits a Python file, trains a tiny model, logs the loss, then the loop suggests improvements. Run it overnight and see what optimizations emerge. (GitHub: cosmopunk/nanochat)
  • Test GPT-5.4 Computer Control: Grab the new API and try having it fill out a form or navigate a website screenshot. Start simple: "Log into this website and download the CSV file." Then graduate to "Book me a flight with these parameters."
  • Create a PR Reviewer Bot: Use Claude's new Code Review feature via API to automatically scan your GitHub pull requests and flag potential bugs before merging. Good way to see if it catches things your team misses.
